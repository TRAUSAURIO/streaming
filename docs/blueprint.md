## Style Guidelines:

- Primary color: Neon Red (#ff1744) for main titles and CTA buttons.
- Secondary color: Neon Cyan (#00f0ff) for subtle details in icons and underlines.
- Background color: Dark background (#0a0a0a) for an immersive experience.
- Card background: #111111 with a subtle border (#ff003322).
- Hero background: Subtle dark-to-red gradient from #0a0a0a to #1a000a.
- Headline font: 'Orbitron' sans-serif for titles to convey a modern feel. Note: currently only Google Fonts are supported.
- Body font: 'Inter' sans-serif for body text to maintain readability and professionalism. Note: currently only Google Fonts are supported.
- Use clean and modern icons from heroicons or font-awesome.
- Glow effect on hover with shadow rgba(255,23,68,0.5) for red, and rgba(0,240,255,0.4) for cyan.
- Mobile-first design, ensuring optimal viewing experience on smaller screens.
